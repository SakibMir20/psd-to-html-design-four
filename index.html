Admin Panel - T3 Framework

There’s a lot in this highly functional template building tool. Check some of the cool ones that would compel you to use it right away.

Global Setting

The global settings are applied for all styles, themes, layout, responsive, and rtl css no matter the settings are changed in any styles.

Compile LESS to CSS

It's developed with LESS. When customizing a site, we suggest working with LESS files, it will be compiled to CSS files later on.

Theme Settings

The Theme settings allows you to configure themes, logo. The settings can be also configured in the ThemeMagic setting panel

Layout Setting

The framework each style will use one layout. There are 2 parts to be configured: Layout structure and Responsive layout configuration.

ThemeMagic

It is powerful and easy theme customization based on ThemeMagic. You are allowed to add unlimited parameters for customization.

Megamenu

It allows you to display your menu flexibility. Arrange menu items in columns, and rows adding any content to the menu.

Custom Code

The tab includes settings that allow you to add scripts like Google Analytic, Live chat service to any position. You may write custom CSS using the style tag also.

Optimize CSS and JS

Optimize CSS and JS is to improve your site performance by join and compress CSS and JS files so that the total size and number of requests will be decreased.

Off-canvas

The new Off-Canvas sidebar is now independent with Collapse Navigation for small screens. The Off-canvas sidebar displays content from a defined position

Style Overrides

Each style includes its own theme, layout, logo, navigation. When creating new style, you can configure those things and override the default style.

Typography Settings

The typography in templates is mainly based on Google Fonts collection (950+ fonts). This settings area allows you to easily choose font's from the Google Font Directory to your template.

Override Template

You can now easily override Joomla layouts using the Framework. Using the template you may create layout easily from Blog to Magazine, from Corporate to Portfolio, etc.
